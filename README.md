# Kdrama V5

Korean and Chinese Drama streaming plugin for CloudStream.

## Categories

- Latest Kdrama (Trakt — snoak's curated list)
- Korean Drama - Recent
- Korean Drama - Top Rated
- Chinese Drama - Recent
- Chinese Drama - Top Rated
- Korean Movies
- Chinese Movies

## Installation

1. Open CloudStream
2. Go to Settings → Extensions
3. Add Repository: `https://raw.githubusercontent.com/Elvizk/Kdrama/builds/repo.json`
4. Install the "KdramaV5" plugin
